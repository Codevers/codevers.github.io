Css Sprite技术

发表于 2014-03-18 | 更新于 2019-03-26 | 分类于 Css学习 | 评论数： | 阅读次数：

Css Sprite，又被叫做css精灵，是一种试图减少图片http请求的技术，从而让图片加载更快，如果http请求次数过多会影响效率，css sprite适用于大型网站，缺点也是有，一：索取图片的坐标，精确每一个像素，费时费力。二：如果动了其中一个图标的坐标位置，那么很有可能所有的图标会混乱，所以说一旦确定了就不要随便乱动了，当然了，用来作翻转效果那最好了，避免了普通翻转图片加载出现空白现象，废话少说，看实例最明白.

注：在写css的时候，千万不要写完css再来整合sprite图片，不然是场灾难。。
