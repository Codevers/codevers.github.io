Css3:过渡属性

发表于 2014-07-16 | 更新于 2019-03-26 | 分类于 Css学习 | 评论数： | 阅读次数：

相信很多朋友在很多前沿网站都见过这种效果，悬浮或点击某一元素时，目标元素会有个类似于动画的滑动效果，像iPhone的解锁一样，很动感，也很潮流，感觉网站一下子变得高大上了，没错！这就是Css3中基本属性之一：“transition过渡”。

根据W3c标准对transition的描述：

“css的transition允许css的属性值在一定的时间区间内平滑地过渡。这种效果可以在鼠标单击、获得焦点、被点击或对元素任何改变中触发，并圆滑地以动画效果改变CSS的属性值。”

先来看语法结构：

transition: property ||  duration ||  timing-function || delay
当然了，也可以跟其他css3一样，设置多个属性集，以“**，**”分开： 
transition:property || duration || timing-function || delay , property ||  duration || timing-function || delay
总共有四个属性值：属性 || 持续时间 || 时间函数 || 延时时间 ## 一：属性（property） property是指元素的属性改变时开始执行transition效果，就是说指定一个需要过渡的属性值，具体的取值如下： * **none**：无效果。 * **property-name**：元素的属性名，当执行过渡效果会以perproty-name为准。 * **all**：默认效果，当指定为all是，任何一个属性的变换都会产生过渡效果 ## 二：持续时间（duration） 这里的持续时间，说白了就是元素进行过渡时，在一个时间段内完成所有过渡效果，值越小，过渡的频率越快，值越大，过渡的频率越平缓，单位为“s(秒)”或者“ms(毫秒)”，与其他属性值一样，可以设置两个time值，用逗号分隔。 ## 三：时间函数（timing-function） 时间函数，说简单点就是过渡的效果，它是根据时间的推进改变属性值的变换频率，具体的如下： * **ease（逐渐缓慢）**：使元素在进行过渡时，变换频率为逐渐变慢的效果。 * **linear（匀速）**：元素在进行过渡时，变换频率为一个匀速效果。 * **ease-in（加速）**：元素在进行过渡时，变换频率为加速模式。 * **ease-out（减速）：**元素在进行过渡时，变换频率为减速模式。 * **ease-in-out（加速后减速）**：元素在进行过渡时，变换频率为开始为加速，将要结束时，减速进行过渡。 * **cubic-bezier（自定义一个赛贝尔时间曲线）**：自定义时间曲线，有四个值：（x1,y1,x2,y2），所有值必须在0-1之间，不然无效，由于没有特殊情况，一般是不用的，所以这里不作介绍了。感兴趣的朋友请查阅相关资料。 ## 四：延时时间（delay） 就是专门指定元素开始过渡的时间，比如说我想在鼠标悬浮元素后，过2秒再进行过渡效果，这里设置delay即可，具体取值如下: * **delay : 2sms** （取值为“s”或“ms”这没什么可说的，很明白） 最终，这些简单的属性值简单的介绍完了，其实这些取值都是有简写方法，形式如下： * **transition：**all 1s ease-in-out 0s; （这里的all指的是所有元素属性都可以进行过渡）。 ### less写法： 
.transition(@property;@duration;@timeFunction;@delay)
{
 -moz-transition:all @duration @timeFunction @delay;
 -webkit-transition:all @duration @timeFunction @delay;
 transition:all @duration @timeFunction @delay;
 }
引用：**.transition**(all , 1s , ease-in-out , 0s)； ### 为了能够保持兼容模式，最好写上各大浏览器的前缀： 
-webkit-transition：all 1s ease-in-out 0s;
-moz-transition：all 1s ease-in-out 0s;
transition：all 1s ease-in-out 0s;
看下浏览器之间的兼容视图： [![2014-07-16_090650](http://www.webfuns.me/webfuns/wp-content/uploads/2014/07/2014-07-16_0906501.jpg)](http://www.webfuns.me/webfuns/wp-content/uploads/2014/07/2014-07-16_0906501.jpg) ### 介绍完了，最后来点简单的实例，加深印象： Html Code : 
<div class="demo"></div>

Css Code :

.demo{
-webkit-transition:all 1s ease 0s;
-moz-transition:all 1s ease 0s;
transition:all 1s ease 0s;
width:100px;
height:100px;
background:#c00;
}

.demo:hover{
margin-left:50px;
}

如果没猜错的话，当鼠标悬浮目标元素后，由于目标元素有一个过渡属性，所以当目标元素发生变换时，都会形成过渡的效果，这里目标元素会进行一个向右缓慢移动的效果，由于这里使用了all值，所以大多的属性都能进行过渡。

好了，算是简单的介绍完了，这里只作了简单的介绍transition过渡，许多的动画效果都可以根据transition效果来完成，比如元素缓慢放大，颜色变慢等等一些基础效果。
