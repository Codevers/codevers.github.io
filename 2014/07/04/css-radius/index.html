Css3:圆角技术

发表于 2014-07-04 | 更新于 2019-03-26 | 分类于 Css学习 | 阅读次数：

圆角技术，早在css2.0中就有所应用，其实现的方法大多是以图片来代替，比如说使用四个圆角图片通过定位来获得圆角，利用多标签配合background和border来实现..等等，今天这里对这些不做介绍，今天讲的是一种更高级的方法，是的，大名鼎鼎的Css3的圆角技术，使用非常广泛，这里作下简要介绍，并且配合几个实例进行更加直观的认识。

首先，css3中的圆角技术看起来很简单，只需要设置些值就能生成圆角了，但殊不知，它可以有更多的设置方法（注：为了节省代码和空间，各大浏览器的前缀名俺就不写了^^）_

其实border-radius和border边框设置一样，都是有四边来组成的，并且可以单独把四边拆开进行单独设置，border-radius拆分如下：

是不是很奇怪？我进行了单独设置为什么会有两个值，其实是这样，第一个值为水平方向的圆角半径，第二个值为垂直方向的圆角半径，第二个值可以不做设置，只设置第一个值的话，那么这个角就为四分之一的角，如果其中一角的值为0，那么直接就是90度的直角。

第一个值为水平的40px，就是说四个角的水平值为40px，第二个为垂直的值为20px，拆开就是：border-radius：40px 40px 40px 40px / 20px 20px 20px 20px；如果值都为一样，那么就可以简写形式：border-radius:40px;即为四角的半径都为40px，

基于上面所说，我简要的介绍了圆角的生成过程，下面说下半径的不同对值：

如果拆分的话是这样的：

水平有两个取值，即为：40px和15px

垂直有两个取值：即为：20px和50px

每一个角的半径都有两种值，水平和垂直，对应的取值范围如下：

  • left-top——botton-right right-top——botton-left
    这是它们的对应关系，跟border边框十分相像，都是遵从TRBl的原则，比如说：border-radius : 20px 30px 10px;

那么botton-left（左下角）对应的是top-right ，即为：30px
