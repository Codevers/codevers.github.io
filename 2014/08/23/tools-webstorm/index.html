WebStorm 8 + live Eidt + JetBranis IDE Support 自动刷新

发表于 2014-08-23 | 更新于 2019-03-26 | 分类于 前端工具 | 评论数： | 阅读次数：

用WebStorm编辑器实现自动刷新页面的功能，不用手动F5去刷新，浏览器插件配合编辑器会自动帮你实现刷新，感觉蛮爽的，下面是具体实现步骤：

1：首先得要有WebStorm编辑器吧，可以去官网下载个。

2：安装完成后，暂时先不管它，打开Chrome浏览器，去扩展程序商店安装JetBranis IDE Support插件。

3：Jet安装完成后，进入编辑器，随便新建一个html页面，在视图（View）里面把live Edit打上勾。

4：WebStorm 端口和JetBrains IDE Support端口保持一致为默认端口 63342。

5：开始调试页面，打开webstorm的debug功能，点击即可。

具体步骤：

view -> live eidt 已打勾
file -> settings -> debugger - > javascript -> built-in server port :63342
JetBrains IDE Support -> 选项 -> Host:127.0.0.1 Port: 63342

最后，调试页面的时候，在编辑器中打开debug功能，即那个界面右上角虫子按钮，点住就能打开自动刷新的Tab页面，（alt+f2启动浏览器）
