Css:制作小三角的三种方法

发表于 2014-04-27 | 更新于 2019-03-26 | 分类于 Css学习 | 评论数： | 阅读次数：

浮动层的小三角效果，在web网页中经常被用到，特别是运用在弹出框中，带个小三角会更为生动，那么如何来制作这些效果，用图片？恩，不错，兼容很好，而且很容易使用，但是有个死点，那就是不灵活，能随意更改颜色吗，能随意创建更多的三角吗，不能，所以下面将会介绍几种用Css制作小三角的方法，更为灵活的，扩展效果。

解决方案一：

为什么对border-style设置dashed虚线就能实现IE6的透明了？其实与其说是支持透明，不如说是让那个三地方消失了！ 在IE6中，点线dotted或dashed(虚线)的是以边框宽度border-width为基准的，点线宽度必须是边框宽度的3倍以上，虚线宽度必须是边框宽度的5倍以上。 公式如下：dashed>=border-width5、dotted>=border-width3，否则点线与虚线将不会显示！ 正是巧妙的运用IE6这一特点bug，才得以让IE6支持一个三角。

解决方案二：

这里改了些参数，不再设置border-style属性，而是利用border-width:0让不需要出现的地方直接为0；想象一下：一个box分为四个独立的三角形，border-top为10，border-right为10px，border-bottom为0；border-left为10px；这样设置就会形成一个缺一角的标签页码似的形状，也就是下面示例中第三个形状，由于border-left 和 right不可见，但却存在(border-width原因)，而border-top却是可见的，并且是有颜色信息的，受到左右两边的排列挤压，自热而然的就出现了个倒立的三角。
